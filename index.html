MARS MapMaker

Follow README to Demo

MARS MapMaker is a client side ReactJS application that was created to work in tandem with MARS from CIRDLES. The purpose of this map maker is to allow geologists to properly register their rock samples with SESAR (System for Earth Sample Registration) in order to aquire their IGSN (International Geo Sample Number). This is accomplished by providing a user interface which allows the user to map the various fields of their metadata to the corresponding field in the SESAR database. After the map is complete, a javascript mapping file is created to automate the process for obtaining the IGSN. This project is currently being tested by Columbia University.

Moodle Testing Framework

Moodle is a free and open-source learning management system written in PHP and distributed under the GNU General Public License. In order to combat existing bugs in Moodle, my team and I set out to produce a testing framework to help Moodle developers better identify some of the potential issues in their code. We designed the framework so that, if a developer is interested in testing a specific method within a class, they would have to do a few things. First, add a test case to the test case folder that follows the requirements of the testing template. After adding the test case, the developer would also create a simple driver in the "testCasesExecutables" folder that follows the driver template and voila! We wanted to build a framework that was both intuitive and made the testing of classes and methods easy.
